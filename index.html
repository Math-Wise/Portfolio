RSA Encrypter

Website

The RSA encryter that I wrote in SAGEmath inputs a file and exports a encrypted file. RSA is primaily used to store passwords in a way that no one can access them. RSA works by converting message first into ASCII. ASCII is a widely known key to convert messages into numbers. Now that the file is one long number, RSA works taking two large primes, n and m and then multiplying (n-1)(m-1). The resulting number is stored along with another number, e that that divids it. These two numbers are you encrytion key. Anyone with these numbers and encrypt messages, but not decrypt them. The ASCII message is broken down into strings of length one less then (n-1)(m-1). Let each string be represented as A1 to An, with i between 1 and n. At this point each one of the ASCII strings, A1 to An are placed into the the formula Ei=Ai^e(mod N) where Ei is the encryted version of Ai. Then all of the Ei are placed into a vector. This vector is the encryted message. For more information on the RSA encrypter or the decrypter click here.
